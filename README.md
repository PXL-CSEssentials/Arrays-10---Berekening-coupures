# Oefening 10: Berekening coupures

In deze toepassing wordt bepaald hoe een bedrag uitgeven kan worden met
zo min mogelijk briefjes/munten. Na de invoer van het bedrag, kan de
gebruiker op de knop "Berekenen" klikken om weer te geven hoe het
betreffende bedrag cash betaald kan worden met zo min mogelijk
eurobriefjes en euromuntstukken.

De knop "Wissen" maakt het bedrag en resultaat terug leeg en legt de
focus op "Bedrag".
